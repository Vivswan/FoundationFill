Foundation Fill

一个用于填充文本字段的 Chrome 扩展，提供预定义模板和使用 LLM API 生成文本的功能

概述

Foundation Fill 帮助您快速使用自定义模板填充文本字段。它特别适用于：

  • 软件工程师创建样板代码片段
  • 作家为 AI 工具使用一致的提示
  • 支持团队使用标准化回复模板
  • 研究人员为 AI 助手使用特定指令

功能

模板管理

创建、编辑、复制、重新排序和管理可重用模板，方便快速访问。

特定域名模板

配置模板仅在特定网站上显示，以获得更好的组织管理。

模板变量

在模板中使用 {{变量:默认值}} 语法，实现自定义值。

右键菜单集成

在任何文本字段上右键单击即可快速访问您的模板。

主题支持

选择浅色/深色/系统主题，并提供多种强调色选项。

多语言支持

使用英语、简体中文或繁体中文界面。

API 生成

通过自定义 API 集成生成文本，生成过程中显示动画效果。

开始使用

安装

从 Chrome 网上应用店安装（推荐）

  1. 访问Chrome 网上应用店页面
  2. 点击"添加至 Chrome"
  3. 接受权限

从 GitHub 发布页安装

  1. 访问GitHub 发布页
  2. 下载最新的发布 zip 文件
  3. 将 zip 文件解压到一个文件夹
  4. 前往 chrome://extensions/
  5. 启用开发者模式
  6. 点击"加载已解压的扩展程序"并选择解压后的文件夹

手动安装（开发）

  1. 克隆仓库或下载源代码
  2. 运行 npm install 安装依赖
  3. 运行 npm run build 构建扩展
  4. 前往 chrome://extensions/
  5. 启用开发者模式
  6. 点击"加载已解压的扩展程序"并选择 dist 目录

使用指南

创建模板

  1. 点击浏览器工具栏中的 Foundation Fill 图标
  2. 点击"新建"按钮创建新模板
  3. 输入您的系统提示和用户提示
  4. 您的模板将自动保存

使用模板

  1. 在浏览网页时右键点击任何文本字段
  2. 从"Foundation Fill"下的右键菜单中选择您的模板
  3. 模板内容将被插入到文本字段中

组织模板

  1. 在扩展弹出窗口中，使用手柄图标拖放模板
  2. 默认模板将始终保持在顶部
  3. 使用复制按钮（复制图标）创建任何模板的副本
  4. 复制的模板将直接放置在原始模板之后

特定域名模板

  1. 编辑模板并点击"域名"按钮
  2. 点击"添加当前域名"或手动输入域名
  3. 模板将只在指定的网站上显示
提示： 使用域名通配符（例如，*.example.com）匹配所有子域名。

使用模板变量

  1. 在模板中使用 {{变量:默认值}} 格式添加变量
  2. 使用模板时，系统会提示您提供值
  3. 具有相同名称的变量将在整个模板中使用相同的值

使用 API 生成文本

  1. 点击齿轮图标进入设置
  2. 输入您的 API 配置（URL、密钥、模型）
  3. 使用"测试提示"按钮通过模板生成文本

设置

API 配置

配置您首选的 LLM API 用于文本生成：

  • 基础 URL： API 端点（例如，https://api.openai.com/v1）
  • API 密钥： 您的认证密钥（安全存储）
  • 模型： 模型标识符（例如，gpt-4）

主题设置

自定义扩展的外观：

  • 主题： 浅色、深色或系统（跟随浏览器的主题）
  • 强调色： 选择蓝色、红色、绿色、紫色、橙色或粉色
  • 语言： 选择英语、简体中文或繁体中文

导入/导出

备份和恢复您的模板和设置：

  • 导出所有数据： 将您的模板和设置保存为 JSON 文件
  • 导入数据： 从之前导出的文件恢复

架构

Foundation Fill 遵循具有面向服务组件的模型-视图架构：

核心组件

  • 模型： 处理数据和业务逻辑（模板、设置）
  • 视图： 渲染用户界面并处理用户交互
  • 服务： 提供可重用功能（存储、API、主题）
  • 后台脚本： 管理右键菜单和消息传递
  • 内容脚本： 填充文本字段并管理文本插入

数据流

模板存储在 Chrome 存储中，由后台脚本访问以填充右键菜单。当选择模板时，如果需要，会处理变量，然后发送到内容脚本以插入到活动文本字段中。
